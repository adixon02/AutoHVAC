AutoHVAC — Smart HVAC Layout Generator
Overview
AutoHVAC is an intelligent HVAC layout and permitting tool that generates building-code-compliant ducted and ductless HVAC system designs based on architectural blueprints or basic project inputs. It instantly performs zoning-accurate load calculations (Manual J), proposes optimal duct paths or mini-split placements, and outputs multiple tiered layouts—from permit-ready to luxury optimization.

Who It’s For
HVAC contractors & engineers needing faster, compliant design docs

Homebuilders & developers looking to simplify HVAC coordination

Architects integrating mechanical into early design phases

HVAC product reps providing white-glove quotes for clients

DIY homeowners or designers preparing for HVAC installs or permits

How It Works
Enter ZIP code

Automatically applies local zoning rules, energy codes, and permit requirements

Upload blueprints (or manually enter key layout info if unavailable)

Blueprint parsing includes floor plans, ceiling heights, room labels, and mechanical room placement

AutoHVAC analyzes the structure and climate zone to output:

Full Manual J load calculation by room/zone

Ducted or ductless layout options (flexible across all brands)

BTU sizing and equipment recommendations based on room demand and system goals

Line set routing, vent placement, and trunk sizing for forced air systems

Select output level:

Permit Plan – minimum viable system to pass inspection

Mid-Tier Plan – optimized for performance and budget

Luxury Plan – most comfortable, efficient, and silent experience possible

Download CAD-ready files (PDF, DXF, SVG) + compliance checklist

Key Features
Brand-agnostic system generation (works with Carrier, Mitsubishi, Daikin, LG, etc.)

Smart zoning based on climate, structure, and ZIP code

Manual J engine integrated into the workflow

Optional Manual D (duct sizing) and Manual S (equipment selection) modules

Export-ready for AutoCAD, Revit, and local permit offices

Future: API for permit platforms, 3D visualization, SKU-level quoting

The Goal
To build the world’s first plug-and-play HVAC planning tool that empowers contractors, builders, and designers to go from floor plan to code-approved HVAC system in minutes—not days. Save time, reduce cost, improve comfort.
